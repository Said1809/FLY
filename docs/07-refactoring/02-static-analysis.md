# Отчёт статического анализа

## Инструмент

`flutter analyze` с набором правил `flutter_lints` (`analysis_options.yaml`).

## Запуск

```bash
flutter analyze
```

## Типичные замечания (категории)

| Категория | Описание | Действие |
|-----------|----------|----------|
| `unused_import` | Неиспользуемые импорты | Удалить |
| `prefer_const_constructors` | Константные конструкторы | По желанию |
| `use_build_context_synchronously` | context после async | Проверить mounted |
