# Матрица стейкхолдеров

| Стейкхолдер | Интерес | Влияние | Ожидания | Стратегия взаимодействия |
|-------------|---------|---------|----------|--------------------------|
| **Студент (автор проекта)** | Высокий | Высокое | Сдать курсовой проект, получить оценку | Основной исполнитель; ведёт документацию |
| **Преподаватель (руководитель)** | Средний | Высокое | Соответствие методичке, архитектура, документация | Регулярные отчёты по этапам, защита |
| **Студенты-пользователи (ЦА)** | Высокий | Среднее | Удобное планирование учёбы | UX-тестирование, сбор обратной связи |
| **Supabase (поставщик BaaS)** | Низкий | Среднее | Корректное использование Auth API | Следование документации, лимиты бесплатного тарифа |
| **Команда Flutter/Dart** | Низкий | Низкое | Использование SDK по назначению | Обновление зависимостей, `flutter doctor` |
| **Комиссия на защите** | Средний | Высокое | Демонстрация работы, ответы на вопросы | Презентация, APK, полный комплект docs |

## Матрица «Влияние — Интерес»

```
         Высокий интерес
              │
   Информировать│ Управлять тесно
   (Flutter)    │ (Преподаватель, Автор)
              │
──────────────┼──────────────  Высокое влияние
              │
   Мониторить  │ Держать в курсе
   (Supabase)  │ (Студенты-пользователи)
              │
         Низкий интерес
```
