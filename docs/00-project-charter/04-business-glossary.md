# Бизнес-глоссарий

Единая терминология предметной области (20 терминов). На Этапе 1 глоссарий
расширяется системными терминами.

| # | Термин | Определение |
|---|--------|-------------|
| 1 | **Задача (Task)** | Единица планирования: название, описание, дата, опциональное время, флаг выполнения. |
| 2 | **Список задач** | Совокупность задач, привязанных к одной календарной дате. |
| 3 | **Дата выполнения** | Календарный день, на который запланирована задача. |
| 4 | **Время задачи** | Опциональный момент в течение дня (часы и минуты). |
| 5 | **Выполненная задача** | Задача с установленным флагом `isCompleted = true`. |
| 6 | **Активная задача** | Невыполненная задача на выбранную дату. |
| 7 | **Неделя** | Семидневный интервал с понедельника по воскресенье. |
| 8 | **Недельный календарь** | Горизонтальная полоса из семи дней с возможностью свайпа между неделями. |
| 9 | **Месячный календарь** | Сетка дней месяца с цветовой индикацией наличия и статуса задач. |
| 10 | **Статус дня** | Состояние даты: `none` (нет задач), `active` (есть невыполненные), `completed` (все выполнены). |
| 11 | **Планирование** | Процесс создания и организации задач во времени. |
| 12 | **Учебное планирование** | Использование приложения для фиксации дедлайнов, занятий и домашних заданий. |
| 13 | **Персонализация** | Настройка темы, шрифта и акцентного цвета интерфейса. |
| 14 | **Светлая тема** | Режим отображения с светлым фоном (`ThemeMode.light`). |
| 15 | **Тёмная тема** | Режим отображения с тёмным фоном (`ThemeMode.dark`). |
| 16 | **Пользователь** | Зарегистрированный участник системы с учётной записью Supabase. |
| 17 | **Сессия** | Состояние аутентификации пользователя после успешного входа. |
| 18 | **Локальное хранилище** | База данных SQLite на устройстве (`todo.db`). |
| 19 | **Офлайн-режим** | Работа с задачами без интернета (данные в SQLite). |
| 20 | **Кроссплатформенность** | Запуск приложения на Android, Web, Windows и других платформах из одной кодовой базы. |
