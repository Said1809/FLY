# Расширенный глоссарий (22 термина)

Бизнес-глоссарий (Этап 0) дополнен системными и техническими терминами.

| # | Термин | Определение |
|---|--------|-------------|
| 1 | **Flutter** | UI-фреймворк Google для кроссплатформенной разработки на Dart. |
| 2 | **Dart** | Язык программирования, используемый во Flutter-приложении. |
| 3 | **Provider** | Пакет управления состоянием; `ChangeNotifier` + `context.watch/read`. |
| 4 | **ChangeNotifier** | Класс Flutter для оповещения слушателей об изменении состояния. |
| 5 | **Repository** | Паттерн абстракции доступа к данным (`TaskRepository`). |
| 6 | **SQLite** | Встраиваемая реляционная СУБД; файл `todo.db` на устройстве. |
| 7 | **sqflite** | Flutter-плагин для работы с SQLite. |
| 8 | **Supabase** | BaaS-платформа; в проекте используется модуль Auth. |
| 9 | **AuthGate** | Виджет-маршрутизатор: HomePage или AuthScreen по наличию сессии. |
| 10 | **SharedPreferences** | Key-value хранилище для настроек UI. |
| 11 | **table_calendar** | Виджет месячного календаря в `TaskCalendarDialog`. |
| 12 | **Material Design 3** | Дизайн-система Google, база UI-компонентов. |
| 13 | **Entity** | Слой доменных объектов (`Task`, `Week`). |
| 14 | **Presentation** | Слой UI: pages, widgets, dialogs. |
| 15 | **Control** | Слой контроллеров (`TaskController`, `WeekController`). |
| 16 | **Model** | Слой репозиториев и моделей настроек. |
| 17 | **Foundation** | Инфраструктура: форматирование дат, сервисы. |
| 18 | **CRUD** | Create, Read, Update, Delete — базовые операции над задачами. |
| 19 | **FAB** | Floating Action Button — кнопка «+» на главном экране. |
| 20 | **PageView** | Виджет постраничной прокрутки; используется для недель. |
| 21 | **ISO 8601** | Формат хранения даты в БД (`YYYY-MM-DD`). |
| 22 | **Tree-shaking** | Оптимизация сборки: удаление неиспользуемых иконок из APK. |
