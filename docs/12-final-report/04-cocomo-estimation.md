## 2. Класс проекта

**Organic** — один разработчик, зрелые фреймворки (Flutter, Supabase).

| Параметр | a | b | c | d |

## 3. Расчёт

**Трудоёмкость:**
```
PM = 2.4 · (2.0)^1.05 ≈ 2.4 · 2.07 ≈ 5.0 чел.-мес.
```

**Длительность:**
```
TDEV = 2.5 · (5.0)^0.38 ≈ 2.5 · 1.90 ≈ 4.8 мес.
```

**Численность:**
```
N = 5.0 / 4.8 ≈ 1.0 чел.
```

## 4. Сопоставление с фактом

| Показатель | COCOMO | Факт |
|-----------|--------|------|
| Трудоёмкость | ≈ 5.0 чел.-мес. | 1 исполнитель |

| Численность | ≈ 1.0 чел. | 1 чел. |

Оценка согласуется с выполнением проекта одним студентом в отведённый срок.

отсутствием собственного backend-сервера.
