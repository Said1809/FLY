# Руководство пользователя

## 1. Назначение приложения

**Fly** — приложение для персонального и учебного планирования. Позволяет создавать
задачи с привязкой к дате и времени, просматривать их в недельном и месячном
календаре, отмечать выполнение и настраивать внешний вид интерфейса.

## 2. Требования к устройству

- **Android:** 8.0 и выше (APK).
- **Web:** современный браузер (Chrome, Edge).
- **Windows:** Windows 10/11 (desktop-сборка).
- **Интернет:** нужен для регистрации и входа; задачи доступны офлайн после входа.

## 3. Начало работы

### 3.1. Регистрация

1. Запустите приложение — откроется экран входа.
2. Переключитесь на режим «Регистрация».
3. Введите email и пароль.
4. При необходимости подтвердите email (зависит от настроек Supabase).
5. После успешной регистрации выполните вход.

### 3.2. Вход

1. Введите email и пароль.
2. Нажмите «Войти». При успехе откроется главный экран с сегодняшней датой.

Сессия сохраняется: повторный вход при следующих запусках не требуется до выхода.

## 4. Главный экран

| Элемент | Назначение |
|---------|------------|
| Шапка (DateHeader) | Текущая дата, кнопки настроек и календаря |
| Полоса недели | Свайп влево/вправо — другие недели; tap на день — выбор даты |
| Список задач | Активные и завершённые задачи на выбранный день |
| Кнопка «+» (FAB) | Создание новой задачи |

## 5. Работа с задачами

### 5.1. Создание

1. Нажмите «+».
2. Заполните название (обязательно), описание, дату и при необходимости время.
3. Нажмите «Сохранить».

### 5.2. Просмотр и удаление

1. Нажмите на карточку задачи.
2. В диалоге отображаются название, описание, дата и время.
3. Кнопка «Удалить» удаляет задачу безвозвратно.

### 5.3. Отметка выполнения

Нажмите checkbox слева от задачи. Выполненные задачи перемещаются в секцию
«Завершённые задачи». Повторное нажатие снимает отметку.

## 6. Календарь

### 6.1. Недельная навигация

Свайпайте полосу недели для перехода к прошлым и будущим неделям.
Нажмите на число дня для загрузки задач этой даты.

### 6.2. Месячный календарь

1. Нажмите иконку календаря в шапке.
2. Дни с задачами подсвечены:
   - **акцентный цвет** — есть невыполненные задачи;
   - **светлый акцент** — все задачи выполнены;
   - **белый** — задач нет.
3. Нажмите на день — календарь закроется, откроется выбранная дата.

## 7. Настройки

Откройте через иконку шестерёнки в шапке.

| Параметр | Варианты |
|----------|----------|
| Тема | Светлая / Тёмная |
| Шрифт | Roboto, Impact, Saxonia Antiqua |
| Основной цвет | 5 предустановленных цветов |

Настройки сохраняются на устройстве автоматически.

## 8. Выход

В настройках нажмите «Выйти». Потребуется повторный вход при следующем запуске.
**Задачи остаются на устройстве** в локальной базе данных.

## 9. Ограничения

- Задачи не синхронизируются между телефоном и компьютером.
- При удалении приложения локальные задачи будут потеряны.
- При смене учётной записи на том же устройстве список задач общий (локальная БД).
