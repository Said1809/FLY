# ADR-002: Паттерн Repository для задач

## Статус

Принято

## Контекст

Контроллеры не должны знать детали SQLite. Нужна абстракция для тестирования.

## Решение

Интерфейс `TaskRepository` + реализация `SqliteTaskRepository`.
Дополнительно — `InMemoryTaskRepository` для модульных тестов.

## Последствия

- (+) Единая точка доступа к задачам
- (+) CRUD инкапсулирован в репозитории
- (−) Дополнительный слой абстракции
