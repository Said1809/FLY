# ADR-003: Supabase Auth для аутентификации

## Статус

Принято

## Контекст

Требуется регистрация и вход пользователей без разработки собственного сервера.

## Решение

Использовать Supabase Auth (email/password). Инициализация в `main.dart`.
Маршрутизация через `AuthGate` и `onAuthStateChange`.

## Последствия

- (+) Готовая инфраструктура auth, без backend на Java/Spring
- (+) Сессия и refresh-токены из коробки
- (−) Зависимость от внешнего сервиса и интернета при первом входе
