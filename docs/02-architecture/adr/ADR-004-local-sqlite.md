# ADR-004: Локальное хранение задач в SQLite

## Статус

Принято

## Контекст

Задачи должны быть доступны офлайн. Облачная синхронизация — вне scope MVP.

## Решение

SQLite через `sqflite`, файл `todo.db`, таблица `tasks`.
Данные не привязаны к `user_id` (одно локальное хранилище на устройство).

## Последствия

- (+) Быстрый доступ, работа без сети
- (+) Простая схема для курсового проекта
- (−) Нет переноса задач между устройствами
- (−) При смене пользователя на том же устройстве задачи общие
