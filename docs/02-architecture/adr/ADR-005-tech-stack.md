# ADR-005: Технологический стек Flutter

## Статус

Принято

## Контекст

Траектория курсового проекта — кроссплатформенная разработка.

## Решение

| Компонент | Технология |
|-----------|------------|
| UI | Flutter 3.x, Material Design 3 |
| Язык | Dart ^3.10 |
| Состояние | Provider + ChangeNotifier |
| Локальная БД | sqflite |
| Auth | supabase_flutter |
| Настройки | shared_preferences |
| Календарь | table_calendar |
| Локализация | intl (ru) |

Целевые платформы: **Android** (APK), **Web**, **Windows**.

## Последствия

- (+) Одна кодовая база для нескольких платформ
- (+) Богатая экосистема пакетов pub.dev
- (−) Размер APK ~54 MB (release)
